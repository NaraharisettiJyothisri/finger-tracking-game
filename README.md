# 🎮 Finger Tracking Game

An interactive browser-based game that uses hand tracking technology to control gameplay using finger movements.

## 🚀 Features

* Real-time hand tracking
* Interactive gameplay
* Browser-based experience
* Computer vision integration
* Responsive user interface

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* Hand Tracking / Computer Vision

## 🎯 Project Objective

To build an interactive game that allows users to control actions using finger movements and hand gestures.
